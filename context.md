# 🧠 주요 결정 사항 및 기술적 배경 (Context)

## ✔️ 핵심 결정 사항 (Decisions)
* **다크 모드 & UI 통일성:** 모든 UI는 프리미엄 다크 모드를 유지하며, React + shadcn/ui 컴포넌트를 활용해 가독성 높은 인터페이스를 유지한다.
* **로컬 파일 백업 (오프라인):** 데이터 관리의 불안함을 막기 위해 `JSON` Import/Export 기능을 제공한다.
* **PWA (Progressive Web App):** 구글 잼처럼 사용자가 앱 형태로 PC에 다운로드(설치)하여 사용할 수 있게 지원한다.
* **버전 관리 전략:** GitHub(깃허브)를 통해 메인 `main` 브랜치에 코드를 백업 및 저장하고 관리한다.
* **가격 엔진 보안장치:** 절대로 [최소 보장 마진율] 이하로 가격 인하 치킨 게임을 벌이지 않도록 강력한 예외 처리(Circuit Breaker, Fail-safe)를 넣는다.
